# Security Policy

## Supported Versions

Security fixes are provided for the latest release and the current `main`
branch.

## Reporting a Vulnerability

Report suspected vulnerabilities privately to
[hello@sabahattinkalkan.com](mailto:hello@sabahattinkalkan.com).

Include:

- affected version or commit;
- reproduction steps;
- expected and observed behavior;
- potential impact;
- any suggested mitigation.

Do not include active credentials, personal data, or production database
contents. Please allow reasonable time to investigate and coordinate a fix
before public disclosure.
